# Github-Contribution-NewYear
Make github contribution say `HAPPY NEW YEAR`

## Steps:
- Copy github home page source code.
- Clone the repo.
- Search for `COPY FROM HERE` and `COPY TO HERE`. 
- Copy the section.
- Paste it in suitable location in copied source code.
